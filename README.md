A Scheme Machine
================

Warning: Completely Incomplete.

Supported syntax:

  * Parenthesis
  * symbols
  * numbers: integers (as `int64_t`) and floats (as `double`)
  * apostrophe: `'foo` as shorthand for `(quote foo)`

Supported operators:

  * `if`
  * `define`  (values, not functions)
  * `set!`
  * `cons`
  * `car`
  * `cdr`
  * `lambda`
  * `begin`
  * `quote`
  * `+`
  * `-`
  * `*`
  * `/`
  * `=`
  * `!=`
  * `<`
  * `>`
  * `<=`
  * `>=`
